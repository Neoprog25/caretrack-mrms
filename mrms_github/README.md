# CareTrack MRMS

Medical Records Management System — BTEC Unit 25 Full Stack Development

## Tech Stack
- Backend: Node.js (vanilla HTTP server)
- Frontend: HTML, CSS, JavaScript
- Auth: Bcrypt + Base64 token
- Data: JSON files
